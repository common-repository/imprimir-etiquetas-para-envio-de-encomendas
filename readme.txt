Imprima etiquetas de envio de encomendas diretamente do WooCommerce, com layout otimizado para folhas A4, cada uma com 12 etiquetas de 105 x 48 mm.

== Description ==

O plugin **Imprimir Etiquetas para Envio de Encomendas** permite imprimir etiquetas de envio diretamente da página de encomendas do WooCommerce, facilitando a gestão logística da sua loja online. O layout das etiquetas está otimizado para folhas A4 com 12 etiquetas (cada uma com 105 x 48 mm), oferecendo uma solução rápida e eficiente para o envio de encomendas.

As etiquetas incluem a morada do cliente e da loja com o logotipo, para garantir uma apresentação profissional nas suas encomendas.

**Principais funcionalidades:**

* Impressão rápida de etiquetas de envio para encomendas, incluindo morada do cliente, morada da loja e logotipo.
* Layout otimizado para 12 etiquetas por folha (A4), cada uma com 105 x 48 mm.
* Seleção da linha de início da impressão, ideal para reutilizar folhas de etiquetas parcialmente usadas.
* Configuração fácil diretamente na página de encomendas do WooCommerce.
* Modal de ajuda com instruções detalhadas sobre o processo de impressão e configuração da impressora.

== Instalação ==

1. Faça o upload dos ficheiros do plugin para o diretório `/wp-content/plugins/imprimir-etiquetas-para-envio-de-encomendas`, ou instale o plugin diretamente através do painel do WordPress.
2. Ative o plugin através do menu "Plugins" no WordPress.
3. Certifique-se de que o WooCommerce está instalado e ativo.
4. O plugin está pronto para uso. Basta aceder à página de encomendas e clicar no botão "Imprimir Etiqueta de Envio".

== Frequently Asked Questions ==

= Qual o tamanho das etiquetas suportadas pelo plugin? =

O plugin está otimizado para etiquetas de tamanho 105 x 48 mm, dispostas em folhas A4, com 12 etiquetas por folha.

= Como configuro a impressão das etiquetas? =

**No Windows:**

No painel de impressão, configure o "Tamanho do papel" para A4 e desmarque a opção "Cabeçalhos e rodapés". Verifique a orientação da folha na impressora para garantir que as etiquetas sejam impressas corretamente.

**No macOS:**

No diálogo de impressão, selecione "A4" como o tamanho do papel. Nas opções de impressão, desmarque a caixa "Imprimir cabeçalhos e rodapés". Ajuste a orientação do papel conforme necessário, dependendo da sua impressora.

= Posso reutilizar folhas de etiquetas parcialmente usadas? =

Sim, o plugin permite selecionar a linha de início para a impressão, possibilitando reutilizar folhas que já tenham sido parcialmente usadas.
